# ATM-Simulation (Python)
This is a beginner-level ATM simulation project written in Python.
It includes:
- Password protection
- Balance chcek
- Deposit money
- Withdraw money
- Transaction history

## How to Run
1. Clone the repository:
  '''bash
  git clone https://github.com/inoplanetyanka24/ATM_Simulation.git
2. Navigate to the project folder:
  '''bash
  cd ATM_Simulation
3. Run the program:
  '''bash
  python atm_simulation.py
